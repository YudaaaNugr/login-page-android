# Build APK dari HP Android

1. Buat repository baru di GitHub.
2. Upload seluruh isi folder project ini, termasuk folder `.github`.
3. Buka tab **Actions** pada repository.
4. Pilih workflow **Build Android APK**.
5. Tekan **Run workflow**.
6. Setelah selesai, buka hasil workflow dan unduh artifact **login-page-apk**.
7. Ekstrak ZIP artifact lalu instal `app-release.apk`.

Catatan: Android mungkin meminta izin **Install unknown apps** untuk browser atau aplikasi File Manager yang dipakai.
