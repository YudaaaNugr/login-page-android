# Login Screen Demo (Flutter)

Halaman login bergaya gelap berdasarkan video referensi pengguna.

## Keamanan
- Tidak terhubung ke server.
- Tidak menyimpan atau mengirim email/password.
- Tombol sosial dan register hanya menampilkan pesan demo.

## Menjalankan
1. Instal Flutter dan Android Studio.
2. Buka terminal di folder proyek.
3. Jalankan:

```bash
flutter create .
flutter pub get
flutter run
```

## Membuat APK

```bash
flutter build apk --release
```

APK akan berada di:
`build/app/outputs/flutter-apk/app-release.apk`
